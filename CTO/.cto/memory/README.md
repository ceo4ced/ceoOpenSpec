# CTO Memory

This directory contains the working memory for the CTO agent (non-SpecKit).
SpecKit memory is in `.specify/memory/`.

## Files

| File | Purpose |
|------|---------|
| `ceo-brief.md` | Brief from CEO with assignment |
| `vendor-evaluations/*.md` | Vendor assessment documents |
| `tech-decisions.md` | Technical decision log |
| `accessibility-audit.md` | Accessibility compliance status |

## SpecKit Memory

For development-specific memory, see:
```
CTO/.specify/memory/
├── constitution.md  # Technical principles
├── spec.md          # Feature specifications
├── plan.md          # Implementation plans
└── tasks.md         # Task breakdowns
```
