# Patent Bar Study AI

An AI-powered study guide and preparation system for the USPTO Patent Bar (Registration Examination).

## Overview

Patent Bar Study AI helps aspiring patent practitioners prepare for the USPTO Registration Examination by providing:

- **AI-Powered Practice Questions** - Adaptive questioning based on your knowledge gaps
- **MPEP Reference Integration** - Quick access to relevant Manual of Patent Examining Procedure sections
- **Spaced Repetition Flashcards** - Efficient memorization of key concepts and rules
- **Progress Tracking** - Detailed analytics on your study progress and readiness
- **Mock Examinations** - Full-length practice exams simulating real test conditions

## Quick Start

```bash
# Install dependencies
pnpm install

# Start the application
pnpm dev
```
