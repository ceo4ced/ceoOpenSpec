# CPO Memory

This directory contains the working memory for the CPO agent.

## Files

| File | Purpose |
|------|---------|
| `ceo-brief.md` | Brief from CEO with assignment |
| `decision-log.md` | All product decisions with rationale |
| `roadmap.md` | Current 3-horizon roadmap |
| `metrics-tree.md` | North star and supporting metrics |
| `initiatives/*.md` | One-pagers for each initiative |

## Subdirectories

### initiatives/
Individual one-pagers for each product initiative

### prd/
Detailed PRDs (when needed for complex features)
