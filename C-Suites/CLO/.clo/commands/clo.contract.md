# clo.contract

## Preamble

This command drafts contract templates for attorney review. **Never for direct use without legal review.**

**Must read before execution:**
1. `.mission/agent-governance.md`
2. `CLO/.ethics/ethics.md`

## Outline

Generate contract drafts from templates with appropriate disclaimers.

## Execution Flow

1. **Identify contract type**
   - What relationship?
   - What's being exchanged?
   - What's the duration?

2. **Gather key terms**
   - Parties
   - Compensation
   - Responsibilities
   - Term and termination

3. **Generate draft**
   - Use appropriate template
   - Customize for specifics
   - Include all disclaimers

4. **Flag review points**
   - High-risk provisions
   - Jurisdiction-specific issues
   - Industry requirements

5. **Require attorney review**
   - NEVER provide final contract
   - ALWAYS mark as DRAFT
