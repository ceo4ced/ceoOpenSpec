# Agent Dashboard - Cloud Run Application

This document specifies the web dashboard for monitoring all C-suite agents.

---

## Overview

The dashboard is a web application running on Cloud Run that provides:
- Real-time view of all agent activity
- Per-position KPI dashboards
- Token usage and cost tracking
- Thinking log visibility
- GREENLIGHT/HALT controls

---

## Technology Stack

| Component | Technology |
|-----------|------------|
| Frontend | Next.js 14 (App Router) |
| Styling | Tailwind CSS |
| Auth | Google SSO via NextAuth.js |
| Data | BigQuery via REST API |
| Real-time | WebSocket for live updates |
| Hosting | Cloud Run |
| Charts | Recharts |

---

## Authentication

### Google SSO Only

```typescript
// app/api/auth/[...nextauth]/route.ts
import NextAuth from 'next-auth'
import GoogleProvider from 'next-auth/providers/google'

export const authOptions = {
  providers: [
    GoogleProvider({
      clientId: process.env.GOOGLE_CLIENT_ID!,
      clientSecret: process.env.GOOGLE_CLIENT_SECRET!,
    }),
  ],
  callbacks: {
    async signIn({ user }) {
      // Whitelist specific emails
      const allowedEmails = process.env.ALLOWED_EMAILS?.split(',') || []
      return allowedEmails.includes(user.email || '')
    },
  },
}
```

### Access Control

```typescript
// Allowed users stored in Secret Manager
ALLOWED_EMAILS=founder@example.com,partner@example.com
```

## BigQuery Queries

### Agent Status Query

```sql
SELECT 
  agent_position,
  COUNT(*) as conversation_count,
  SUM(total_tokens) as total_tokens,
  SUM(total_cost) as total_cost,
  MAX(completed_at) as last_activity
FROM `{project}.{dataset}.conversations`
WHERE 
  business_id = @business_id
  AND DATE(created_at) = CURRENT_DATE()
GROUP BY agent_position
```

### Thinking Logs Query

```sql
SELECT 
  log_id,
  step_number,
  thinking_type,
  content,
  confidence_score,
  timestamp
FROM `{project}.{dataset}.thinking_logs`
WHERE 
  business_id = @business_id
  AND agent_position = @position
  AND DATE(timestamp) = CURRENT_DATE()
ORDER BY timestamp DESC
LIMIT 50
```

### Cost by Agent Query

```sql
SELECT 
  agent_position,
  model_name,
  SUM(total_cost) as cost,
  SUM(total_tokens) as tokens
FROM `{project}.{dataset}.conversations`
WHERE 
  business_id = @business_id
  AND created_at >= TIMESTAMP_SUB(CURRENT_TIMESTAMP(), INTERVAL 7 DAY)
GROUP BY agent_position, model_name
ORDER BY cost DESC
```

## Real-Time Updates

### WebSocket Connection

```typescript
// hooks/useAgentUpdates.ts
export function useAgentUpdates(businessId: string) {
  const [updates, setUpdates] = useState<AgentUpdate[]>([])
  
  useEffect(() => {
    const ws = new WebSocket(
      `${process.env.NEXT_PUBLIC_WS_URL}/updates/${businessId}`
    )
    
    ws.onmessage = (event) => {
      const update = JSON.parse(event.data)
      setUpdates(prev => [update, ...prev].slice(0, 100))
    }
    
    return () => ws.close()
  }, [businessId])
  
  return updates
}
```

### Update Types

```typescript
type AgentUpdate = {
  type: 'conversation_start' | 'conversation_end' | 'thinking' | 'message' | 'escalation'
  agent: string
  timestamp: string
  data: any
}
```
