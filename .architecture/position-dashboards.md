# Position Dashboards

## Overview

Each C-suite position has a dedicated dashboard view showing metrics, activity, and controls relevant to their domain.

> **Chairman Dashboard** aggregates ALL data across all positions. Position dashboards show role-specific views.

## Dashboard Implementation Notes

### Technology Stack
- **Frontend**: React/Next.js or vanilla HTML/CSS/JS
- **Charts**: Chart.js, Recharts, or D3.js
- **Real-time**: WebSockets or Server-Sent Events
- **Hosting**: Cloud Run

### Data Sources
All dashboards pull from BigQuery as the single source of truth.

### Refresh Rates
- Financial data: Real-time
- Operational metrics: 60-second refresh
- Analytics: Hourly
- Compliance: Daily

### Access Control
Each position can only see their own dashboard unless they're the Human (Chairman), who sees everything.
